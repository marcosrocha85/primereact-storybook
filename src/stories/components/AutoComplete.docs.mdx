## Basic

Type at least one character to filter the suggestion list.

<BasicDemo />

<Source
  code={`const [value, setValue] = useState(null);
const [suggestions, setSuggestions] = useState([]);

<AutoComplete
  value={value}
  field="name"
  suggestions={suggestions}
  completeMethod={(event) => setSuggestions(filterCountries(event.query))}
  onChange={(event) => setValue(event.value)}
  placeholder="Search country"
/>`}
  language="tsx"
/>

## Float Label

Wrap the component in `p-float-label` to have the label float above the input on focus or when a value is selected.

<FloatLabelDemo />

<Source
  code={`<span className="p-float-label">
  <AutoComplete
    inputId="float-autocomplete"
    value={value}
    field="name"
    suggestions={suggestions}
    completeMethod={(event) => setSuggestions(filterCountries(event.query))}
    onChange={(event) => setValue(event.value)}
  />
  <label htmlFor="float-autocomplete">Country</label>
</span>`}
  language="tsx"
/>
